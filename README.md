<h2>👆 Como instalar</h2>

```bash
  # Clone o projeto e entre no diretório
  $ git clone https://github.com/pauloelf/pokenext.git
  $ cd pokenext
```
```bash
  # Instale as dependências
  $ npm install
  ---/ ou /---
  $ pnpm install
```
```bash
  # Inicie o projeto
  $ npm run dev
  ---/ ou /---
  $ pnpm run dev
```
